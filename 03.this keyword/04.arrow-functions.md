# Arrow functions

## 1. Concise syntax

- No need for function keyword.
- If we have single argument, we can omit ().
- If the body has a single expression, you can omit the curly braces and return keyword.

Traditional function:

``` javascript

const add = function(a, b) {
  return a + b;
};

```

Arrow function:

``` javascript

const add = (a, b) => a + b;

```
